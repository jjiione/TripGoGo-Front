# :clipboard: 주요 기능
<h5> :loudspeaker: 기본 기능</h5>   

1. Main Page  

2. 카테고리 별 관광지 조회(관광지 정보 조회)  

3. 회원 관리 페이지  

4. 로그인 / 로그아웃 페이지    

<h5> :loudspeaker: 추가 기능</h5>  

1. MyTrip(나만의 여행 계획 페이지)  

2. 여행 정보 공유 게시판  

3. 길 찾기
